# Images Directory

Place your images here:

## Required Images

1. **profile-photo.jpg** (or .png) - Your profile photo for the About section
   - Recommended size: 300x300 pixels (square)
   - Format: JPG or PNG

## Optional Images

- **hero-bg.jpg** - Background image for hero section
- **project-screenshots/** - Screenshots for projects

## Usage

After adding your photo, the portfolio will automatically display it in the About section.
